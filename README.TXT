PROJECT TITLE:  A Transposition cipher.

PURPOSE OF PROJECT: The text sent by you may be interfered by some intruder,so inorder to protect the message from the intruder its important to encrypt the message and send it so that it cannot be interpreted by the intruder.At the reciver end the it must be decrypted back to original text so that it can be read by the reciever. 

HOW TO START THIS PROJECT:  The program is written in java,compile and run it  using JAVA runtime environment provided by oracle.  
USER INSTRUCTIONS:  Provide the string input and then enter 'e' to encrypt or 'd' to decrypt the entered string.
